版面设计

网站的风格与色彩选择：主页页面以白色为背景，黑白灰三色为主体。子页面多用图片作为背景，字体用黑、白、灰色给人一种清析的感觉。所有页面色调统一，布局相似。

版面分栏结构，即导航栏、轮播图、相关展示内容、设计说明与调查问卷、版权信息。

在内容展示区又分为了5个板块，分别展示了重庆的气候地理文化、重庆历史、重庆的夜晚、重庆的麻辣火锅以及重庆为什么被称为科幻之城。除了第一个板块外，其它每个板块都有一个按钮，点击按钮后会跳转到各个板块对应的子页面，在这些页面中有更加详细的说明。

在主页面介绍重庆的气候地理文化这个板块采用文字与图片左右排列的方法，将重庆的气候地理文化做了一个简单的概括，可以让用户对重庆有一个初步的认识

在介绍重庆历史这个板块，采用一张重庆火车站的黑白画作为背景，整体色调以灰色和白色为主，辅以红色作为点缀，以左边文字右边图片的形式介绍了重庆从石器时代到2019年的历史，条理清晰，可以让用户对重庆历史有一个比较概括的了解。

在介绍重庆的夜晚这个板块，采用一张重庆的照片作为背景，整体色调以蓝色为主，符合重庆夜晚深邃，时尚的气质。在具体介绍时，采用文字与图片相结合的模式，介绍了重庆几个观看夜景的观赏地，并且贴心地准备了观赏的小提示，为想要去重庆旅游的用户提供了不错的参考。

在介绍重庆的麻辣火锅这个板块，采用一张重庆火锅的照片作为背景，十分贴合主题，先通过图片与文字相结合的方式介绍了重庆火锅的主要特点，然后在最下方用一个轮播图展示重庆火锅的主要菜品，模仿用户在火锅店点餐时的场景。

设计亮点

响应式布局

响应式设计可以向用户提供友好的Web界面,同样的布局,却可以在不同的设备上有不同排版。

独特的文字排版

展示了重庆的气候地理文化时利用不规则的文字排版设计，来说明主题，从而彰显低调且时尚的设计感，让网页更加饱满。

同色系色彩的巧妙搭配

单一的纯色作为背景难免会显得有些枯燥，也会给人严肃的感觉，因此在制作子页面时我利用一些色系比较接近的色彩来进行搭配，使网页设计更加柔和协调，丰富和优化黑白网页背景，同时也能让用户觉得不难么沉闷。

技术细节

响应式设计

1. 设置 Meta 标签。

2.通过媒介查询来设置样式 Media Queries。

3.可以使用bootstrap，通过设置col-来进行响应式开发。

使用swiper轮播图

1.点击进入swiper官网下载swiper压缩包。

2.在head标签中引入css文件，路径记得更改;在body标签中引入js文件。

3.在官网复制代码。

滚动监听

1.向想要监听的元素（通常是 body）添加 data-spy="scroll" 。

2.然后添加 data-target 属性，它的值为导航栏的 id 或 class (.navbar)。这样就可以联系上可滚动区域。

3.注意可滚动项元素上的 id 必须匹配导航栏上的链接选项 。

3

最终测试

1.查看页面的Title是否准确和合理。

2.LOGO图片显示是否正确。

3.首页登录、注册的功能是否实现。

4.栏目下的文章标题、图片等链接是否正确。

5.查看文章详细页面的内容是否存在乱码、页面样式是否统一。

6.各个页面的样式风格是否统一,各个页面的大小是否一致,各个页面的Title是否正确。

7.栏目名称、文章内容等处的文字是否正确，有无错别字或乱码；同一级别的字体、大小、颜色是否统一；字体显示是否有问题，大小是否统一，版式是否合理。

8.切换窗口大小，将窗口缩小后，页面是否按比例缩小或出现滚动条。

9.各个页面缩小的风格是否一致，版式是否合理；按钮大小是否一致。
